# scala-lsp

Scala language server for Claude Code, providing code intelligence features like go-to-definition, find references, and error checking.

## Supported Extensions

`.scala`, `.sc`, `.sbt`

## Installation

Install the Metals language server via Coursier:

```bash
cs install metals
```
